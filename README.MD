App Name: TODO List iOS

###### Description
The goal of this project is to create an app for iOS that would allow the user to create, edit, update their records.

###### Must Have Features
* Display Tasks List
* Create A New Task
* Edit Task
* Delete Task

###### Nice-to-have Features
* Swipe to delete
* iCloud synchronization
* Due date reminders
* Reorder list
* Search by query
